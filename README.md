# DFIR Playbooks

Practical Digital Forensics and Incident Response (DFIR) playbooks designed for
Security Operations Center (SOC) and Incident Response teams.

These playbooks focus on **real-world execution**, not theory.

## Coverage
- Phishing incidents
- Malware infections
- Ransomware response
- Insider threat investigations
- Cloud security incidents

## Design Principles
- SOC-ready and actionable
- Aligned with NIST and SANS methodologies
- Clear decision points
- Minimal tooling assumptions

## Incident Response Phases
Each playbook follows:
1. Detection
2. Triage
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

## Intended Audience
- SOC Analysts (Tier 1–3)
- Incident Responders
- Blue Team engineers
