# Contributing to DFIR Playbooks

Thank you for your interest in contributing.

This repository focuses on **practical, real-world DFIR and incident response playbooks**
used by SOC and Incident Response teams.

## Scope
We welcome contributions related to:
- Incident response playbooks
- DFIR workflows
- Triage, containment, eradication, and recovery steps
- Lessons learned and improvement suggestions

## Contribution Requirements
- Use clear and concise language
- Focus on actionable steps, not theory
- Avoid sensitive or confidential data
- Do not include exploit code or offensive tooling
- Keep content vendor-neutral where possible

## Pull Request Guidelines
- One playbook or improvement per pull request
- Clearly describe what was added or improved
- Ensure formatting matches existing playbooks

Low-quality, theoretical, or copy-pasted content may be rejected.

## Code of Conduct
All contributors are expected to be respectful and professional.
